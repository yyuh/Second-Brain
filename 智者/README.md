# 智者 — 第二大脑子模块

## 这是什么

一个持续内化思维模型的 AI Skill。41 条模型覆盖投资、成长、认知、AI 时代判断四大领域。

## 怎么用（自己用）

1. 把整个 `智者/` 目录复制到你的 Agent 技能目录
2. 重启 Agent，它将自动加载 41 条思维模型

## 怎么用（卖出去）

1. 压缩 `智者/` 目录 → `智者-v4.1.zip`
2. 发给买家，让对方放到技能目录
3. 后续更新：你更新 Second Brain，重新导出 `智者/`，发给买家

## 维护者使用

规范源在 `../wiki/insights/思维模型.md`。更新流程：
1. 改 思维模型.md → 2. 更新 core/models.json → 3. 跑 sync.bat → 完成

## 目录结构

```
智者/
├── SKILL.md          # 技能定义
├── core/
│   ├── models.json   # 模型库（派生自 ../wiki/insights/思维模型.md）
│   └── graph.json    # 模型连接网络
└── reports/          # 趋势报告
```
