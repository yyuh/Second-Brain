# 项目总览

当前进行中和计划中的项目。

## 进行中

### 智者skill（内容提炼服务）
- **类型**：AI服务/API
- **描述**：预训练+服务化的内容提炼引擎。输入原始文字（视频/文章），AI按学习的风格输出结构化提炼（摘要/金句/观点/行动建议）
- **技术方案**：RAG + Prompt（推荐），System Prompt + Few-shot, 向量数据库（Chroma/Pinecone）检索成功人士语料
- **API设计**：`POST /api/wisdom/refine`，传入原文+平台+长度限制
- **状态**：设计阶段，需要准备30-50组"原文→提炼结果"的训练数据
- **来源**：DeepSeek对话 [2026-06-15-安卓应用开发咨询.md]

### 小伴 APP
- **类型**：Android APP
- **技术栈**：Kotlin + OkHttp + Gson + Coroutines + DeepSeek API + ASR语音输入
- **描述**：有温度的AI伙伴，用户记录想法，AI感性、主动回复。支持文字输入和语音输入
- **项目结构**：完整的Android项目已在DeepSeek对话中生成，含布局文件、Adapter、Manifest等
- **状态**：代码已生成，可打包APK
- **来源**：DeepSeek对话 [2026-06-15-点子督促助手开发.md]

### 富爸爸棋盘游戏
- **类型**：微信小程序
- **描述**：模拟现金流的棋盘游戏，基于《富爸爸穷爸爸》理念
- **技术栈**：微信小程序原生（WXML + WXSS + JS）
- **特色功能**：
  - 每日新闻弹窗
  - 每日财经新闻采集 cron（每天 8:00 自动抓取）
  - 业务贷款、破产救助贷款等财务机制
- **开发工具**：CodeBuddy + 微信开发者工具
- **当前版本**：`rich-dad-game - 副本` 目录
- **状态**：活跃开发中
- **最新进展**：正在办理软件著作权，计划以个人账号先上广告版验证市场（来源：DeepSeek对话 2026-06-15）
